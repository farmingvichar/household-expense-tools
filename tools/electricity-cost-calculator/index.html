How to use

  1. Enter the appliance name if you want it shown in the result.
  2. Add the appliance power in watts from its label or manual.
  3. Enter how many hours the appliance is used per day.
  4. Enter the electricity rate per kWh and the number of days used per month.
  5. Select calculate to view estimated daily, monthly, and yearly cost.

What this calculator includes

Watts means the appliance power rating. Hours per day is the average daily use time. Rate per kWh is the cost of one unit of electricity. Days per month is how many days the appliance is used in that month.

Fixed charges, taxes, slab rates, meter fees, subsidies, and other bill fees are not included. Real electricity bills can be different because utilities may apply extra charges or changing rates.

FAQ

Does this calculator show my exact electricity bill?

No. It estimates appliance energy cost only. Real bills may include fixed charges, taxes, slab rates, meter fees, and rate changes.

What is kWh?

kWh means kilowatt-hour. It is a common unit used to measure electricity consumption.

Can I use this for any appliance?

Yes. You can use it for an appliance if you know its watts, daily usage time, electricity rate, and monthly usage days.

Does this calculator save my data?

No. It does not use localStorage, cookies, accounts, or server submission.
