FAQ

Common questions

Do I need to create an account?

No. The tools can be used without login, account, password, or profile.

Are the tools free to use?

Yes. The current household calculators and tracker are free to use in your browser.

Does the website track personal expenses online?

No. The site does not use login, analytics, tracking scripts, or a backend database. The Daily Expense Tracker saves entries only in your current browser.

Which tools are currently available?

Current tools include Daily Expense Tracker, Grocery Budget Calculator, Food Waste Cost Calculator, Grocery Unit Price Calculator, Laundry Cost Calculator, and Toilet Paper Value Calculator.
