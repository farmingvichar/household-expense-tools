Privacy Policy

Privacy details for Household Expense Tools.

Household Expense Tools is built as a simple browser-based website. The Daily Expense Tracker saves entries in your own browser. The calculators use the numbers you enter only for the current calculation and do not save them.

Last updated: July 3, 2026

Privacy summary

  • No login is required
  • No account is required
  • No analytics are currently used
  • No external ad scripts are currently added

Data collection

What this site collects

Household Expense Tools does not ask you to create an account, sign in, or submit personal profile details to use the tools.

Expense entries

When you use the Daily Expense Tracker, the expense details you enter are saved in your current browser using localStorage. These entries are not sent to a Household Expense Tools backend database.

Calculator values

When you use calculators such as the Grocery Budget Calculator, Food Waste Cost Calculator, Grocery Unit Price Calculator, Laundry Cost Calculator, or Toilet Paper Value Calculator, the numbers you enter are used only in your browser for that calculation. These calculator values are not saved by the site and are not sent to a backend server.

Account data

The site does not currently have user accounts. It does not collect names, passwords, profile photos, payment details, or account login information.

Analytics and tracking

The site currently does not use analytics tools, tracking pixels, behavioral tracking scripts, or third-party advertising scripts.

Contact information

If you choose to contact the site owner later through a listed contact method, the information you send may be used only to read and respond to your message.

Browser storage

How localStorage works

localStorage is a storage area inside your web browser. It helps the Daily Expense Tracker remember your entries on the same device and browser.

Where are my saved expenses stored?

Your saved expenses are stored in the browser you are using. They are not saved in a site account, and they are not synced by Household Expense Tools across your devices.

Can Household Expense Tools see my saved expense entries?

The current tool is designed so your expense entries stay in your browser. Household Expense Tools does not provide a backend account area where those entries are uploaded or reviewed.

Can my browser delete saved entries?

Yes. Saved entries may be removed if you clear browser data, clear site storage, use private browsing, reset your browser, uninstall the browser, or change to another device.

Does exporting data send it to a server?

The current export feature is intended to create a file from the entries already saved in your browser. It does not require sending those entries to a Household Expense Tools server.

Technical details

Hosting, caching, and normal website requests

Household Expense Tools is a static website hosted on GitHub Pages. Like most websites, pages are delivered through standard web requests.

Static hosting

The site does not currently use its own backend database for the Daily Expense Tracker. The page files are served as static HTML, CSS, and JavaScript.

Service worker cache

The website may use a lightweight service worker to help cache existing site files for faster loading or basic offline support. This cache is for site files, not for creating an online account.

Cookies

The site currently does not set a cookie banner because it does not currently use analytics cookies, advertising cookies, or login cookies.

External scripts

No external analytics or advertising scripts are currently added. If this changes in the future, this policy should be updated before those scripts are used.

Advertising

Future advertising notice

Household Expense Tools may apply for advertising programs such as Google AdSense in the future. At this time, no external advertising scripts are added to the site.

If advertising is added later, this Privacy Policy should be updated to explain what advertising partners may collect, how ads may be served, and what choices users may have.

User control

How you can manage your data

The Daily Expense Tracker stores saved expense entries in your browser. The other calculators are designed for one-time calculations and do not save calculator values.

Delete one entry

The Daily Expense Tracker lets you delete individual expense entries when you no longer need them.

Clear all entries

The tool also includes an option to clear all saved expenses from the current browser storage.

Clear browser storage

You can also remove saved site data through your browser settings. The exact steps depend on the browser you use.

Limits

Important privacy and use limitations

The current tool is designed for simple personal organization. It should not be treated as official financial recordkeeping software.

  • Saved entries may be lost if browser storage is cleared.
  • Entries do not automatically sync across devices.
  • The tool does not replace bank records, tax records, invoices, or professional accounting software.
  • The site does not provide financial, tax, legal, investment, or accounting advice.

Updates

Changes to this Privacy Policy

This policy may be updated when new tools, contact options, analytics, advertising, or other site features are added.

When this page is updated, the “Last updated” date should be changed so users can see when the privacy information was last reviewed.
