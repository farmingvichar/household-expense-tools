About this website

Simple household expense tools that run in your browser.

Household Expense Tools is a small collection of browser-based utilities for everyday home spending, grocery costs, kitchen planning, and basic household budget organization.

Current status

  • First live tool: Daily Expense Tracker
  • No login or account required
  • No backend database for saved expenses
  • Entries stay in the current browser

What the site does

Built for basic household organization

The site is made for people who want simple tools without installing a full finance app or creating an account.

Daily expense tracking

The current Daily Expense Tracker lets users add everyday household expenses and review simple totals for today, this month, all saved entries, and total number of entries.

Household-first use

The tools are intended for common home costs such as groceries, food, transport, bills, household items, education, health, and other daily spending.

Lightweight pages

The website is built as a static GitHub Pages site using plain HTML, CSS, and JavaScript. It does not need a server account system to run the current tool.

Free to use

The current tool is free to use in the browser. Advertising may be added later, but no external ad scripts are currently used on the site.

How it works

Your saved expenses use browser storage

The Daily Expense Tracker uses localStorage, which is a storage area inside your web browser.

What does localStorage mean here?

When you add an expense, the entry is saved in the browser you are currently using on the current device. The site does not send those expense entries to a backend database.

Will my entries appear on another device?

No. Because there is no account system or cloud sync, entries saved in one browser will not automatically appear in another browser or on another device.

Can browser data be removed?

Yes. If you clear site data, clear browser storage, use private browsing, reset the browser, or change devices, saved entries may no longer be available.
